Nancy Drew: Secrets Can Kill 
DreamCatcher Interactive    

Below is information to help ensure Nancy Drew: Secrets Can 
Kill runs smoothly on your system.

MINIMUM REQUIREMENTS
* Microsoft Windows 95 or Windows 98 or Windows ME
* 166 MHz Pentium Processor (200 MHz recommended)
* 16 MB of RAM
* 16-bit color graphics video card
* 16-bit Windows-compatible stereo sound card
* 8x CD-ROM Drive
* 42 MB hard drive space using the Compact install
  option or 220 MB hard drive space using the
  Recommended install option.
* Mouse

BACKGROUND APPLICATIONS
It is strongly recommended that you exit all Windows programs 
before playing the game.  This includes applications that run 
in the background like Norton Utilities.

PERFORMANCE
To obtain optimal performance, install the game with the 
'Recommended' setup type.  This installs often-accessed files
to the hard drive.

UNINSTALLING
To Uninstall Secrets Can Kill, select the 'Uninstall Secrets 
Can Kill' option from the Nancy Drew program folder.  Saved
games will remain on the hard drive. You can also uninstall
the game through the Control Panel's Add/Remove Programs area.

TASK SWITCHING
Secrets Can Kill was designed to use all available resources
on your computer, therefore you should NOT switch focus away
from the game using Alt-Tab or Alt-Esc.  Doing so, in some 
cases, might cause your game or your system to crash.

TROUBLE SHOOTING
Game Lockups:     
                  If the game locks up during play, make sure you
                  are using the latest DirectSound drivers for your
                  sound card and that your sound card is configured
                  correctly. 

                  To verify you have the latest DirectSound drivers
                  available for your sound card, contact the sound
                  card manufacturer.  After you verify you are using
                  the latest sound card drivers, configure the sound
                  card according to the manufacturer's instructions.  
                  If necessary, obtain and install the latest DirectSound 
                  drivers for your sound card. You may also need to 
                  configure your sound card's Direct Memory Access 
                  (DMA) settings.

                  If this doesn't work or you cannot obtain the latest
                  drivers, try the recommendation under No Sound.

Choppy Sound:     
                  See Game Lockups above.

No Sound:         
                  If your system produces no sound when trying to run the
                  game, your sound card might not support DirectSound.
                  Set 'UseDirectSound' in the game.ini file to 0.

Video Conflicts with DirectSound:

		  Testing has shown that on some computers there is a 
		  problem with DirectSound when the game changes the 
		  color depth to the required 16-bit color. This happens
		  if the game starts and you hear no sound or the game locks
		  up or exits when you make a selection from the Main Menu.
		  To eliminate this problem, make sure your video card is
		  running in 16-bit mode before you start the game. You can
		  check this by right-clicking the mouse over an empty area
		  on your desktop.  Select "Properties", then "Settings".
		  Make sure the "Color palette" says "High Color" or the 
		  "16-bit" setting.  If you see a different setting such
		  as "256 Color" or "True Color", change the setting to
		  "High Color" or "16-bit". Your computer will then adjust
		  your video card for 16-bit color.  Restart the game.
